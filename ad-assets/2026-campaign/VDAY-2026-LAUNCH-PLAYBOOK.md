## Post-Launch Monitoring

### Days 1-3: Learning Phase
- **DO NOT** make changes
- Let Meta gather 50+ conversions
- Expect CPAs to fluctuate

### Days 4-7: Early Optimization
| CPA | Action |
|-----|--------|
| < $20 | Excellent - monitor |
| $20-30 | Good - hold steady |
| $30-40 | Watch closely |
| > $40 | Review creatives, check ad set breakdown |

### After 50 Conversions: Scaling
| ROAS | Action |
|------|--------|
| > 4x | Increase budget 20% every 48hrs |
| 3-4x | Increase budget 10% every 48hrs |
| 2-3x | Hold, monitor |
| < 2x | Pause underperforming ads |

## Troubleshooting

### Low delivery
- Check ad approval status
- Verify pixel is firing
- Increase budget slightly

### High CPA (>$35)
- Check ad set breakdown - is one dragging?
- Review which creative/copy combos are spending
- Consider pausing worst performers after 7 days

### No conversions
- Verify checkout flow works
- Check pixel events in Events Manager
- Review landing page load time
